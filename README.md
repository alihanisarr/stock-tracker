# Stock Tracker

A React app that tracks real-time stock prices using the Finnhub API.

## Features
- Search any stock by symbol
- Set a refresh interval in minutes and seconds
- Displays a live table of Open, High, Low, Current, and Previous Close prices 


### Prerequisites
- Node.js installed
- A free API key from [finnhub.io](https://finnhub.io)

### Installation

1. Clone the repo: https://github.com/alihanisarr/stock-tracker.git
2. Install dependencies: npm install
3. Create a .env file in the root folder and add your API key
4. Create a .env file and add your API key
5. npm start
